## Testes

- Para executar os Testes Unitários:
```shell
./gradlew test
```
- Para executar os Testes de Integração:
```shell
./gradlew integrationTest
```
- Para executar os Testes Sistemicos:
```shell
./gradlew systemTest
```
- Para executar os Testes de Performance:
```shell
./gradlew performanceTest
```
